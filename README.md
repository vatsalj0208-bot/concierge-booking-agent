# Concierge Booking Agent
Minimal multi-agent capstone project.
Rubric mapping:
- Multi-agent: Concierge -> SearchAgent -> BookingAgent
- Tools: search_tool, booking_tool, memory_tool, logger
- Sessions & Memory: memory.json stores user preferences
- Observability: logs.txt records agent/tool events
- Evaluation: tests/evaluation.md contains manual test cases and results
